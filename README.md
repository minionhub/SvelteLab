**✨ Features:**

- 🌗 Light / Dark Mode
- 📄 SvelteKit File Icons and Scaffolding
- 📒 Templates (TypeScript, Tailwind, Mdsvex)
- 🧹 Formatting
- ⌨️ Vim Keybindings
- 👻 Hide Config Files
- 🎨 Command Palette

🧡 Made with Svelte, for Svelte, by Svelte lovers!

🔌 Powered by SvelteKit, WebContainers, CodeMirror, Xterm.js and PocketBase
